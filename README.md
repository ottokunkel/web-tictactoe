## Getting Started

### Installation

1. **Clone the repository:**

    ```sh
    git clone https://github.com/yourusername/web-tictactoe.git
    ```

2. **Navigate to the project directory:**

    ```sh
    cd web-tictactoe
    ```

3. **Open `index.html` in your web browser:**

    ```sh
    open index.html
    ```
